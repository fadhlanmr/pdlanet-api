# Running the Code Locally

### Prerequisites:

1. Python3
2. Pipenv Modules

### Instructions:

1. Make sure to have pipenv, if not yet avail :
```
pip install pipenv
```
Install depedencies / packages :
```
pipenv install
```
2. Pip environment command :
```bash
pipenv shell
```
4. Run the program with uvicron :
```
uvicorn [script]:app --reload
```
